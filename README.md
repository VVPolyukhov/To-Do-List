Todo Application

Demo - https://vvpolyukhov.github.io/To-Do-List/

-----

**Stack:**

* JavaScript ES6+
* React
* Bootstrap
* Local Storage

-----

**Features:**

* Adding a task to the list
* Removing a case from the list
* Marking a task as important
* Searching by cases
* Filtering by issues (all, completed, active)
* Convenient interface
